# ESP-IDF Examples

This directory contains example applications that demonstrate how to use the Amazon KVS WebRTC SDK with ESP32 devices. Each example showcases different features and use cases of the SDK.

Each example includes its own `idf_component.yml` file that specifies the required dependencies and components.

## Building Examples

To build any of these examples:

1. Navigate to the example directory
2. Run `idf.py build`
3. Flash the binary to your ESP32 device using `idf.py flash`

For more detailed instructions, refer to the README file within each example directory.
