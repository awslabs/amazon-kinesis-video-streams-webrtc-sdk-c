ESP WebRTC Camera SDK
=====================

Welcome to the WebRTC Camera SDK documentation by Espressif. This camera solution enables ESP32 devices to stream real‑time video and audio using WebRTC with flexible signaling backends (KVS, AppRTC, Bridge, or custom) across multiple ESP32 platforms.

**✨ New Simplified API**: Features 4-line configuration with smart defaults, pluggable signaling/peer connection interfaces, and advanced configuration APIs for power users.
